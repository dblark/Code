# DES算法的Pascal实现

>来自博客

## 说明

这是一个用Pascal实现的DES加密系统。
它没有用任何库，这个Windows库的作用只是为了在文件加密中获取文件的长度。
更多可以去[我的博客](http://zhoushengjie.pw/)的[DES](http://zhoushengjie.pw/加密算法/DES算法.html)了解。
